# 13. Roman to Integer
## Description
Given a roman numeral, convert it to an integer.

Input is guaranteed to be within the range from 1 to 3999.

## Translation
给定一个罗马数字，将其转换为整数。 

输入保证在1到3999的范围内。
## Ideas
暴力破解，依次判断输入的字符串是否以千位、百位、十位、个位的罗马数字开头。

然后再依次判断以[1,9]的哪个字符开头。
